Project:Omni-Sense

Intro

Omni-Sense was created as a response to childern and eldery suffering health issues due to radical tempertures and humidity .Omni-Sense is a wearable neckles that is shaped in a form of a box. It contains two sensors for the temperture and humidity. It has two alert sensors, a flashing light and an audio alert.

Specs

Omni-Sense casing is a 3-D printed designed made from PLA. The sesnors onboard ranged from temp/humidity sensors to the alert sensors the flashing light and audio alert. A arduino nano and a esp 8266 web server.

How does it works

The arduino allows commucation between the sesnors, meanwhile the esp8266 allows a web server to be created and create an UI read the incoming data.
